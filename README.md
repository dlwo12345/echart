# echart 프로토타이핑

- 실무 적용 전 프로토타이핑을 위한 프로젝트

## 개발언어

- Angular v10 기반으로 개발되었으며 기본 프로젝트는 angular-cli를 이용.
- [Angular CLI](https://github.com/angular/angular-cli) version 10

## 추가 패키지

- echarts
  > npm i -S echarts

- ngx-echarts
  > npm i -S ngx-echarts

## demo 

https://dlwo12345.github.io/echart

## port

4400(default)

## start

- npm run start

## build

- npm run build

## deploy

- npm run deploy
